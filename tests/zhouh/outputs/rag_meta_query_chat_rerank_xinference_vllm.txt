在RUIJIERG-S6000E系列交换机上，配置一定范围的接口可以使用`interface range`命令。该命令格式如下：

```
interface range [type] device/slot/{第一个 port} - {最后一个 port}
```

其中，`type`可以是FastEthernet、GigabitEthernet、TenGigabitEthernet、FortyGigabitEthernet（取决于你的设备支持的速率），AggregatePort（聚合端口）、VLAN ID范围或Loopback ID范围，或者Tunnel ID范围。每个接口类型之间的分隔以逗号(,)进行。

例如：
- 如果你想配置某个槽位的所有Fast Ethernet端口，你可以输入：`interface range FastEthernet device/slot/1 - 24`
- 对于千兆以太网端口： `interface range GigabitEthernet device/slot/15 - 30`

请确保所有指定的接口属于同一类型，并且port号码的有效范围内。

2. 然后启用EEE功能。使用命令`eee enable`来开启端口节能模式。如果需要取消该设置，可以使用`no eee enable`或默认恢复设置。

3. 缺省情况下，EEE为关闭状态，所以启用时可能需要这个操作。

4. 如果你的设备支持LPI（低功耗空闲）模式，在闲置时间让端口进入低功耗状态以节省能源，请确认设备和所连接的网络设备都支持并协商此特性。
