在RUIJIERG-S6000E系列交换机上，配置一定范围的接口可以使用`interface range`命令。该命令的格式如下：

```
interface range [type] device/slot/{第一个 port} - {最后一个 port}
```

其中，`type`可以是FastEthernet、GigabitEthernet、TenGigabitEthernet、FortyGigabitEthernet、AggregatePort、VLAN、Loopback或Tunnel，具体取决于你想要配置的接口类型。例如，如果你想要配置某个FastEthernet段，命令可能是：

```
interface range FastEthernet device/slot/1-10
```

或者配置一个VLAN范围：

```
interface range vlan 100-200
```

```
errdisable recovery interval time
```

其中，`interval time` 是您想要设置的自动恢复定时时间，取值范围为30秒到86400秒（1天）。例如，如果设置为10分钟，命令将如下：

```
errdisable recovery interval 600
```

这将使端口在违例发生后按照设定的时间自动恢复。请注意，缺省情况下，端口违例不会自动恢复，此命令需要手动配置。
